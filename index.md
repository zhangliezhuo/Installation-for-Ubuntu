### 更新软件仓库列表

更新系统内的软件仓库列表，获取最新的软件列表

```markdown
sudo apt-get update
```

然后需要更新本地的软件，下面这行命令会将系统所有软件与列表比对并且更新

```markdown
sudo apt-get upgrade
```
